## 🎥 撮影Tips

### 準備
- [ ] ローカル環境またはデプロイ先でアプリを起動
- [ ] ウォレットに十分なSUIトークン（Testnet）
- [ ] ブラウザをフルスクリーンまたはウィンドウモード（1920x1080推奨）
- [ ] 不要なブラウザ拡張を無効化
- [ ] 撮影用のダミーデータを準備（Moment 2-3個）

### 撮影設定
- **解像度**: 1920x1080（フルHD）
- **フレームレート**: 30fps以上
- **音声**: 48kHz（音声解説する場合）
- **マウスカーソル**: 強調表示ONにする（Loom等）

### 撮影時の注意
1. **動作をゆっくり**: クリックやスクロールはゆっくり見やすく
2. **待機時間**: ページ遷移やローディングは2-3秒待つ
3. **強調**: 重要な部分にマウスカーソルを置く
4. **カット編集**: 不要な待機時間は後で編集
5. **音声なしOK**: 字幕だけでも十分伝わるように

### 編集（オプション）
- 不要な待機時間をカット
- 重要な部分にテキスト挿入（「Walrus統合」など）
- BGM追加（著作権フリー音源）
- トランジション効果（控えめに）
